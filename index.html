Tentang

CaptionAI adalah ekstensi Chrome yang menghasilkan closed caption otomatis untuk konten video berbahasa Indonesia. Dirancang khusus untuk membantu pengguna tunarungu dan gangguan pendengaran agar dapat menikmati konten video tanpa hambatan.

Panduan
1

Download file extension

Klik tombol download di bawah dan simpan file captionai-extension.zip ke komputer kamu.

2

Ekstrak file ZIP

Klik kanan file ZIP → pilih Extract All → pilih folder tujuan → klik Extract.

3

Buka halaman ekstensi Chrome

Di Chrome, ketik chrome://extensions di address bar → aktifkan Developer mode di pojok kanan atas.

4

Muat ekstensi

Klik Load unpacked → pilih folder hasil ekstrak → klik Select Folder. Icon CaptionAI akan muncul di toolbar.

5

Mulai gunakan

Buka video YouTube berbahasa Indonesia → klik icon CaptionAI → klik Mulai Caption. Caption akan muncul di layar.
